# Sample Reports

This folder contains sanitized sample reports generated by CyberProbe during real security investigations and posture assessments. They demonstrate the reporting capabilities of the platform across different domains.

## Reports

| Report | Description |
|--------|-------------|
| `blast_radius_choke_points_2026-02-26.html` | Blast radius analysis identifying choke points, lateral movement paths, and high-impact devices in the environment |
| `cnapp_end_to_end_scenario_2026-04-12.html` | End-to-end CNAPP scenario covering cloud-native application protection from code to runtime |
| `cnapp_shift_left_scenario_2026-04-12.html` | Shift-left security scenario with DevSecOps findings, vulnerability posture, container security, and compliance |

| `standards_compliance_report_2026-04-13.html` | Regulatory compliance posture across industry standards (CIS, NIST, PCI-DSS, ISO 27001) |

## How They Were Generated

These reports were generated autonomously by CyberProbe using GitHub Copilot agent mode with MCP (Model Context Protocol) server integrations:

1. **Data collection** — KQL queries executed against Microsoft Sentinel and Defender XDR via MCP tools
2. **Enrichment** — Threat intelligence from AbuseIPDB, Shodan, IPInfo, and VirusTotal
3. **Analysis** — AI-driven correlation, risk scoring, and MITRE ATT&CK mapping
4. **Rendering** — Self-contained HTML reports with embedded CSS, SVG charts, and Mermaid diagrams

Each report is a single standalone HTML file with no external dependencies.

## How to View

These reports are self-contained HTML files with embedded styles, charts, and interactive elements. GitHub does not render HTML files directly — you need to download them first.

### Option 1 — Download from GitHub UI

1. Navigate to the report file on GitHub
2. Click the **Download raw file** button (↓ icon) in the top-right corner of the file view
3. Open the downloaded `.html` file in Microsoft Edge, Chrome, or any modern browser

### Option 2 — Clone and open locally

```bash
git clone https://github.com/olchar/CyberProbe.git
cd CyberProbe/reports
```

Then open any `.html` file in your browser:

- **Windows**: `start blast_radius_choke_points_2026-02-26.html`
- **macOS**: `open blast_radius_choke_points_2026-02-26.html`
- **Linux**: `xdg-open blast_radius_choke_points_2026-02-26.html`

### Option 3 — Raw URL via GitHub Pages or htmlpreview

You can preview any report without downloading by prepending the raw GitHub URL with [htmlpreview.github.io](https://htmlpreview.github.io):

```
https://htmlpreview.github.io/?https://github.com/olchar/CyberProbe/blob/main/reports/blast_radius_choke_points_2026-02-26.html
```

> **Note**: Some interactive features (e.g., Mermaid diagram rendering) may require a local browser for full functionality.

## Sanitization

All reports have been sanitized before publishing. Tenant IDs, workspace IDs, organization names, and any internal references have been replaced with generic Contoso placeholders. No real PII or internal infrastructure details are exposed.
