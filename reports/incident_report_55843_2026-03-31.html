📝 Executive Summary

On March 31, 2026, between 12:34 and 12:50 UTC, an AI agent operating via the MCPPlatform-WebApi Graph API client programmatically uploaded 6 sensitive security documents to the OneDrive of user User E (user5@contoso.com), a designated Critical Asset.

The documents contained active Defender alerts, security threat incidents, and incident reports dated to the same day — indicating the AI agent was extracting live security operations data and writing it to a user's personal cloud storage. Microsoft DLP immediately flagged all uploads as Exfiltration tactic events under the "Block sharing sensitive information in Teams" policy.

Simultaneously, Microsoft Purview Insider Risk Management generated 3 separate High-severity alerts for the same user — "IRM Risky AI Usage - M365 Copilot" and "IRM Policy for Risky AI usage" — confirming that AI-driven activity by or attributed to admin.ops is being monitored as risky.

Critical failure: Two automated response playbooks (Block-EntraIDUser and AWSIAM-DeleteAccessKeys) attempted to contain the incident but both failed due to an expired token. The user account remains fully active and the AI agent pipeline has not been disrupted.

Data Extraction Queries

Query 1 — Incident Discovery (AI/Copilot Keywords)

Feeds: Executive Summary, Correlated Incidents
SecurityIncident
| where TimeGenerated > ago(1d)
| summarize arg_max(TimeGenerated, *) by IncidentNumber
| where Title has_any ("copilot", "AI", "agent", "DLP")
  and (Title has_any ("exfiltration", "exfil", "data loss", "DLP", "sensitive"))
| project IncidentNumber, Title, Severity, Status, Classification, CreatedTime, AlertIds
| order by Severity asc, CreatedTime desc
Result: 12 incidents returned — DLP exfiltration and Copilot label restriction incidents

Query 2 — Exfiltration & IRM Alerts for admin.ops

Feeds: Attack Timeline, Correlated Incidents, Entity Enrichment
SecurityAlert
| where TimeGenerated > ago(1d)
| where tostring(Entities) has "admin.ops" and tostring(Entities) has "ops.contoso.com"
| summarize arg_max(TimeGenerated, *) by SystemAlertId
| project SystemAlertId, AlertName, AlertSeverity, Tactics, ProviderName, ProductName,
    TimeGenerated, ExtendedProperties=tostring(ExtendedProperties), Entities=tostring(Entities)
| order by TimeGenerated desc
Result: 7 alerts — 3x IRM Risky AI Usage (High), 3x DLP Exfiltration (High), 1x DLP Copilot Label (Low)

Query 3 — OfficeActivity File Operations (AI Agent Identification)

Feeds: File Evidence Table, Attack Pattern Diagram, UserAgent Analysis
OfficeActivity
| where TimeGenerated > ago(1d)
| where OfficeWorkload in~ ("OneDrive", "SharePoint", "MicrosoftTeams")
| where Operation in~ ("FileUploaded", "FileAccessed", "FileDownloaded",
    "FileSyncUploadedFull", "FileModified", "FileCopied", "FileMoved")
| where UserId has_any ("admin.ops", "securitycopilotagentuser")
    or SourceFileName has_any ("Project Obsidian", "Security_Threat_Incidents",
       "Security_Incident_Report")
| project TimeGenerated, UserId, Operation, OfficeWorkload, SourceFileName,
    Site_Url=tostring(Site_Url), ClientIP, UserAgent=tostring(UserAgent)
| order by TimeGenerated desc
Result: 20+ records — revealed MCPPlatform-WebApi/1.0 UserAgent on 5/6 file uploads, identifying the AI agent
